## ジャンル
2D横スクロールのエンドレスランゲーム

## ゲーム内容
・Playerが右に向かって自動で走り出しマウスクリックでジャンプが可能

・Playerのアクションのジャンプとジャンプ後の降下で障害物やギミックをよける

・時間経過でスコアを加算

・何度でも挑戦可能

。画面左の外に出る、もしくはギミックに当たるとGameOver

・HPの要素はなし

#追加要素案（余裕があれば）
・敵Enemyの実装

・マップを追加

・スコアの上昇条件を増やす

・ギミックの追加

・エフェクトのアップグレード

## 操作方法
マウスの左クリックのみ
